**Description**

The Banking System is a Java-based application designed to simulate real-world banking operations. The project focuses on implementing core Java concepts along with a graphical user interface to provide an interactive and user-friendly banking experience.

This application allows users to manage bank accounts, perform transactions, and track account activity efficiently through a structured and intuitive interface.

**Features**

Account creation and account management

Deposit and withdrawal functionality

Real-time balance inquiry

Transaction history tracking

Interactive and user-friendly interface built using Java Swing
